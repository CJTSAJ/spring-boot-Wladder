# spring-boot-Wladder
## 关于该项目
- 该项目基于eclips(4.7.2），用spring-boot自动生成。其中去掉了原本的logging日志，改用log4j2。用junit测试了该项目的一个简单函数。
- 项目运行在默认的8080端口，在/ladder路径下调用wordladder程序，并且需要传入3个参数(filename word1 word2)。运行后的程序如图
